Declaración Accesibilidad

Inicio > Declaración accesibilidad

En nuestro sitio web, nos esforzamos por garantizar que el contenido sea accesible para todos los usuarios.

Medidas implementadas

  • Idioma principal de la página: El idioma principal de la página es el español. Está indicado con la etiqueta <html lang="es"> para mayor entendimiento y accesibilidad.
  • Etiquetado semántico: Usamos etiquetas como <header>, <nav>, <main>, <table>, <p>, etc.
  • Texto alternativo de las imágenes: Todas las imágenes incluyen texto alternativo descriptivo con la etiqueta <alt>.
  • Mecanismos de navegación: Navegación accesible con el teclado y estructura de contenido clara. Además de añadir un mapa visual y migas de pan para facilitar la navegación entre las páginas.
  • Adicción de los significados de las abreviaturas y cambios de idioma: Etiquetas para el significado de las abreviaturas (<abbr>) y la indicación de los cambios de idiomas.
  • Implementaciones estilo página web: Modo oscuro para reducir la fatiga visual.
  • Adaptación de los textos: Opción de texto grande para mejorar la legibilidad.
  • Mejora del contraste: Estilos de alto contraste para una mejor visibilidad.
  • Diseño de la página de impresión: Estilos de impresión que eliminan elementos innecesarios.
  • Diseño interactivos que mejoran la atención: Resaltado del foco en elementos interactivos para facilitar la navegación.

Compromiso de accesibilidad

Enciclopedia MUA se compromete a hacer accesible su sitio web de conformidad con el **Real Decreto 1112/2018**, de 7 de septiembre, sobre la accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público.

Cumplimiento normativo

Este sitio web es parcialmente conforme con el **nivel AA de las Directrices de Accesibilidad para el Contenido Web 2.1** debido a las excepciones y falta de conformidad que se indican a continuación.

Contenido no accesible

El contenido que no es accesible se detalla a continuación:

  • Algunos videos pueden no incluir transcripciones textuales.
  • Algunos documentos descargables pueden no estar completamente accesibles.
  • Puede haber fallos ocasionales en el etiquetado de elementos interactivos.

Preparación de la presente declaración

Esta declaración fue preparada el 14/11/2024 y revisada el 16/11/2024. El método empleado para su preparación ha sido una autoevaluación realizada por los desarrolladores del sitio.
